# Demo Script

## 10-second opening

We built [product] for [user] who struggles with [pain]. It helps them [outcome] in [time/metric].

## 3-minute flow

1. Problem: show the pain.
2. Action: perform the primary workflow.
3. Result: show the output.
4. Proof: show the metric or before/after.
5. Production path: explain why this can become real.
